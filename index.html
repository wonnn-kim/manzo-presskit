Quick Facts

Game Name : Manzo

Description:

This game is a hybrid of exploration, metroidvania, and rhythm genres. Horror elements are added to heighten the tension as you dive deeper into the ocean depths.

Developer Contact: weare1842@gmail.com

Features:

  • Limited Lighting
  • Original soundtrack
  • 3 impressive bosses

Links:

  • GitHub Repository

Platforms & Release:

  • PC (Windows)
  • Planned Release: Fall 2025

Gameplay:

Use mouse left click for dash, hover with the mouse, collect fish, sell them, buy modules, and go deeper. Avoid monsters and crashing rocks, endure boss attacks, and continue your adventure.
